# React + TypeScript + Vite

React + TanStack Table ile geliştirilmiş stok yönetim uygulaması.

## Özellikler

- Stok listeleme, ekleme, düzenleme ve silme
- Tekli ve toplu silme (onay modalı ile)
- Arama, depo ve durum filtresi
- Sıralama ve sayfalama
- CSV export (tümü veya seçili)
- Dark / Light mod
